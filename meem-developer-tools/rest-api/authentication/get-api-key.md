# Get API Key



**Endpoint:** `/api/1.0/me/apiKey`

**Authenticated Request:** Yes

**Method:** `GET`

**Response Body**

| Parameter Name | Data Type (required) | Description                                                                            |
| -------------- | -------------------- | -------------------------------------------------------------------------------------- |
| **jwt**        | `string (Y)`         | A JWT token that will not expire and can be used to make requests on behalf of a user. |
